# Disciples-Sacred-Lands-Dbi-codec
Decoding and Encoding in Disciples Sacred Lands

With this script its possible to replace portraits with new images in Disciples Sacred Lands. 
It works only with 55x67 or 115x67 sized PNG images, and can only handle UNIT.DBI
Tutorial video:
https://www.youtube.com/watch?v=5cDO1W_Qvsk

changes: In 2.6 version its possible to encode images with longer bytelength into shorter bytelength images.
